1- CREATION: 
in root folder run: npm create vite@latest client --template vanilla
cd into client and run: npm install
Create styles and import assets
Create frontend on client folder
Create a server folder in the root, cd into it and run: npm init -y
Install dependencies: run npm install cors dotenv express nodemon openai

2- OPENAI 
Go to opeanai website, login and click in profile image > view api keys > create new secret key > copy it, paste into the .env file

Run frontend: cd into client and run: npm run dev
Run backend: cd into server and run: npm run server
